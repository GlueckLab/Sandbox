Welcome to Glimmpse, the general linear multivariate model power and sample size estimator

This tool is designed to aid researchers in selecting appropriate power and sample size for studies using the general linear multivariate model for data analysis. Common examples of this model include the one and two sample t-test, ANOVA, ANCOVA, and regression.

To begin, you will need to create a study design or upload a previously saved design. Select one of the options below to enter the study design wizard.
